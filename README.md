## 简介
> 利用docker技术，去实现家用电视的意见部署

## 涉及服务
| 系统 | 功能 | 汉化 | 说明 |
| :---: | :---: | :---: | :---: |
| [jellyfin](https://jellyfin.org/) | 媒体服务器 | ⭕ | 刮削信息，提供观看服务 |


## 启动服务

```angular2html
git clone https://github.com/al6nlee/auto-TV
cd auto-TV
sh tools/init.sh
sh tools/start.sh
```

## 访问服务
```angular2html
http://127.0.0.1:8096
```
